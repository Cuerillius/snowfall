# Snowfall Userscript
Simple Tampermonkey script that lets it snow on all websites!

## Installation
Make sure you have the Tampermonkey extension installed in your browser.

Then click the link below to install the script automatically.

[Install](https://raw.githubusercontent.com/Cuerillius/snowfall/refs/heads/main/snowfall.user.js)

## Acknowledgements
Most of the snowfall code is from here: [codepen.io/bsehovac](https://codepen.io/bsehovac/pen/GPwXxq), I just tweaked some things and made it work as a Tampermonkey extension.
